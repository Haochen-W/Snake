# Snake Game
This is a simple snake game. 

## Instructions
* Use arrows or WSAD to control the snake.
* Use \"[\" and \"]\" to reduce or increase the speed of motion before the game starts.
* Use \"r\" to restart the game.
* Use space to pause the game.
